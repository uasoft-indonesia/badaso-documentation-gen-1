## CSS

Kamu dapat menambahkan css kustom di direktori `assets` seperti pada direktori di bawah ini.

```
|-- your project
    |-- resources
        |-- js
            |-- badaso
                |-- assets
                    |-- scss
```

## JS

Kamu juga dapat menambahkan javascript kustom pada struktur direktori di bawah ini.

```
|-- your project
    |-- package.json
```
