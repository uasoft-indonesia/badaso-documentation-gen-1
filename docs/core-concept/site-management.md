Pada fitur manajemen situs, pengguna dapat mengubah informasi mengenai situs yang menggunakan Badaso. Informasi yang dapat diubah meliputi:

* Title
* Description (Belum Digunakan)
* Logo
* Warna Header
* Warna Teks Header
